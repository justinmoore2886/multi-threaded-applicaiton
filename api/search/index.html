Type Members

  1.  class InvertedIndex[A] extends AnyRef

    A thread-safe inverted index.

    A thread-safe inverted index. Read-only queries can be done fully in parallel. Modifications are single-threaded, done by a background thread.

    A

    the type of the sources being indexed (file, url, etc.)

  2.  trait Locations extends AnyRef

    A list of word locations.

    A list of word locations. The list contains only positive values, is sorted and is non-empty.

  3.  trait Records extends AnyRef

    An iterator of words and their locations.

  4.  class SafeQueue[A] extends AnyRef

    Mutable thread-safe queue.

    Mutable thread-safe queue. Elements can be added at the back or at the front of the queue. Queues are unbounded (adding operations never block). Instances of this class can safely be shared among threads.

  5.  class Semaphore extends AnyRef

    A simple semaphore.

    A simple semaphore. It is a java.util.concurrent.Semaphore in unfair mode.

    Instances of this class are thread-safe.
